# Article-sentiment-analyzer
AI-based Article Analyzer that extracts content from URLs and provides sentiment analysis with interactive UI and rating system.
## Features
- Extracts article content from URL
- Displays summary and full blog
- Sentiment analysis (Positive, Neutral, Negative)
- Star rating system
- Interactive UI

## Tech Stack
- Python (Flask)
- BeautifulSoup (Web Scraping)
- TextBlob (Sentiment Analysis)
- HTML, CSS

## How to Run
1. Install dependencies:
   pip install flask requests beautifulsoup4 textblob

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
